发表于 2019-02-24 | 分类于 basis

业务场景

typescript 交叉类型

typescript 运行结果

–target ES5, log 方法被拷贝
–target ESNext, log 方法没有被拷贝

场景分析

typescript 编译结果

在 –target ES5 的情况下，编译成 prototype 形式，在 –target ESNext 的情况下，直接编译成 class 格式；

原因

class 格式下将 prototype 的 log 属性的 enumerable 值设置成了 false
